a lightweight markdown editor with git sync. built with:

- React frontend
- Electron wrapper for desktop
- Go backend
- SQLite database

features:
- git based version control
- websocket based live collaboration
- live markdown preview like obsidian (although not half as good)

to run:
cd into the frontend and:

```
npm i
```
```
npm run dev
```

in another terminal
```
go build
```
```
./docsmith
```
application should be on http://localhost:5173/ !
